Installation

Péalable

Vous devez disposer de Java, version minimum 1.7.0_45, pour que oStorybook puisse fonctionner.



  1. Téléchargez le fichier Zip dont le nom est oStorybook-X.y.z.zip.
  2. Extraire ce fichiers dans un répertoire local (je recommande de créer ce répertoire avec le nom oStorybook, vous pourrez ainsi l'identifier plus facilement).
  3. Cherchez le fichier oStorybook.jar ou Storybook.jar dans ce répertoire.
  4. Faites un click droit sur le fichier et un tirez le sur votre bureau; choisissez de créer un racourci.


Valable pour toute distribution basée sur le système de paquetage Debian.

  1. Téléchargez le fichier dont le nom est oStorybook-X.y.z.bin.
  2. Ouvrez une fenre de terminat et executez la commande:
    sh oStorybook-X.y.z.bin
    l'installation se fait par défaut dans le /opt de l'utilisateur.
  3. Vérifiez que le racourci a bien été créé sur votre bureau (si vous utilisez Gnome). Sinon créez un racourci avec votre méthode habituelle.

Pour toute autre distribution Linux utilisez la procédure de base avec le fichier Zip.
