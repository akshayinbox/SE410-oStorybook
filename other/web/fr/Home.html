oStorybook l'un des meilleurs logiciels pour les écrivains

Storybook est un logiciel Open Source pour les écrivains, essayistes, auteurs. Utilisé depuis l'ébauche jusqu'à l'ouvrage final, oStorybook vous permet de ne jamais perdre de vue la trame de l'histoire. oStorybook vous aide à conserver la haute main sur les différents développement de votre récit.

oStorybook vous aide à structurer votre travail

Gérer les personnages, les lieux, les scènes, les objets, les repères les idées se fait dans un ensemble unique et cohérent. Une interface simple permet de définir chaque élément et chaque scène tout en conservant un aperçu permanent de votre travail grâce a des outils comprénhensibles et pratiques. Avec Memoria chaque objet peut être relié aux autres éléments à chaque étape.
